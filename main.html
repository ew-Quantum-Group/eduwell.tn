"Calcule l’intégrale ∫(2x + 3) dx."
Sahl AI :
Pour calculer l’intégrale de 2x + 3, on intègre chaque terme séparément.

∫2x dx = 2 × (x² / 2) = x²
∫3 dx = 3x

Donc :
∫(2x + 3) dx = x² + 3x + C

Où C est la constante d’intégration.
